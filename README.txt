RENDU TP JEE
FOREST - YAKER

Nous avons développé les fonctionnalités demandées pour ce TP.

Le projet ce divise en deux grosses partie :
	- ejb 
	- war 

Le premier point "ejb" contient les éléments côté "serveur" contenant les EJB Command et Livre ainsi 
que les interfaces / classes faisant le lien entre ces ejb, la base de données et la seconde partie war.

Le second point, contient les servlets intégrant les interfaces dites "facades" de l'ejb et traitant les
données et redirigeant l'utilisateur vers les jsp correspodantes. Les jsp qui sont contenus dans cette 
partie également, contenant de l'html et utilisant les données sessions enregistrées via les servlets.
